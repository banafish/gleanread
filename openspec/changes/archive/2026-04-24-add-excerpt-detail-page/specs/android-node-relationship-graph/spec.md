## MODIFIED Requirements

### 需求:局部关系图必须支持手势与点击交互
系统必须允许用户在图谱视图中执行拖拽、缩放、点击聚焦和本地导航操作。

#### 场景:拖拽和缩放图谱
- **当** 用户在图谱视图中拖拽或双指缩放
- **那么** 系统必须允许平移和缩放图谱，而不得关闭当前图谱视图

#### 场景:点击图谱中的知识节点
- **当** 用户点击图谱中的某个知识节点
- **那么** 系统必须聚焦该节点，并允许用户跳转到该节点的详情页

#### 场景:点击图谱中的摘录节点
- **当** 用户点击图谱中的某个摘录节点
- **那么** 系统必须打开该摘录对应的详情页
